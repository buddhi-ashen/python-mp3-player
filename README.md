# python-mp3-player
created by buddhi ashen
In this project we are going to make a mp3 player in python by using pygame and tkinter .

please make sure you have install tkinter and pygame in your python directory 

to install:-

1) tkinter:
  pip install tkinter
  
2) pygame:
    pip install pygame
